# 系統更新紀錄 (v20.3)

**日期**: 2026-03-26

## 系統異動與最佳化 (System Changes)
- **移除不穩定的 PDF 匯出功能**:
  - 由於前端 `html2pdf.js` 與 `Chart.js` 響應式框架相容性極度不佳，反覆發生凍結與跑版問題。在與使用者討論並確認其為「不實用」之後，決定將對應的按鈕、函式以及外部引用宣告全數拔除。
  - 後續本系統將全面專注於已經修復完成之「**匯出 Excel (含圖表)**」功能，作為穩定且實用的報表導出唯一管道。

## 版本更新 (Version Bump)
- **全站版號推進**: 配合本次拔除作業，將網頁標題、頁首與頁尾標示統一推進至 `v20.3`。
